Frequently Asked Questions

Common questions and answers

Is this legal?

Yes, this uses the official Telegram API. However, always respect group rules and privacy. Don't scrape private data without permission.

Do I need to be a group member?

For public groups, no. For private groups, you need to be a member and have the group invite link.

Will I get banned?

Unlikely if you use reasonable limits. Telegram has rate limits which the scraper respects. Don't scrape thousands of groups rapidly.

What file types can I scrape?

Any file type! Common ones: .ex4, .ex5 (MT4/MT5 EAs), .zip, .rar, .pdf, .doc, images, videos, and more.

Can I download the actual files?

This tool exports metadata (filename, size, message link). To download files, use the message links in Telegram.

How do I get API credentials?

Go to my.telegram.org/apps, log in, and create a new application.
